Rules & Server Information

Crime Rules

General expectations around escalation, robbery pacing, scene fairness, and not power-gaming illegal loops.

What it is

This page focuses on Crime Rules inside the Rules & Server Information section. It is written around the live Hallowed Hills resource stack and meant to explain what the system actually does for a player, not just repeat the page title back to them.

How to get started

Start on the main rules page first, then use this page to zoom into the exact policy area that is giving you trouble during a scene. Treat it as clarification for live RP behavior, not as a loophole hunt.

How it works in practice

Expect this topic to affect live scene decisions, escalation, memory, conflict pacing, or fair-play expectations. These pages are written to help players recognize the intended behavior during real RP instead of only after a punishment happens.

What to know before you commit

The most important thing to know is that staff will enforce intent, fairness, and realism, not just literal wording. If a scene choice only works because you are trying to out-lawyer the page, you are already off track.

What this wiki leaves out on purpose

This guide does not publish staff-only tools, exploit-sensitive backend logic, answer strings for gated minigames, or hidden coordinates that are supposed to be discovered through play.
