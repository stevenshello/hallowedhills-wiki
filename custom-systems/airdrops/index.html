Custom Hallowed Hills Systems

Airdrops

The contested event-style reward layer without exposing exact event handling details.

What it is

Airdrops are contested event-style content that create sudden high-interest hotspots in the city. They are built to pull players into a risky reward race instead of a quiet grind.

How to get started

Approach airdrops with awareness, not greed. They are event content, so expect other players, possible conflict, and a need for fast decision-making.

How it works in practice

When an airdrop event hits, it creates a temporary opportunity layer around timing, contest, and reward retrieval. It is meant to feel more like an event scene than a static farm route.

What to know before you commit

The reward is only part of the point. Airdrops also create social pressure, conflict, and opportunistic RP.

What this wiki leaves out on purpose

Exact event timings, spawn locations, and backend event logic stay off the wiki on purpose.
